# 🎬 Movie Recommender System

A Content-Based Movie Recommendation System that suggests similar movies based on plot, genres, cast, and keywords using Machine Learning.

## ✨ Features

- 🔍 Recommend 5 similar movies based on user selection
- 🎨 Display movie posters from TMDB API
- 👍👎 User feedback system (Like/Dislike)
- 🚀 Fast and responsive UI
- 📊 Feedback tracking with CSV storage

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- NLTK
- TMDB API
